## Installation

AwesomeMVVM is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "AwesomeMVVM", :git => 'https://github.com/congnt24/AwesomeMVVM.git'
```
